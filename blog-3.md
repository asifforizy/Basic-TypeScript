How do Generics allow you to build reusable components and functions that stay strictly typed regardless of the data structures passed in?

answer:
Generics let you code one reusable function or component that works with different data types keeping type safety.
you do a type placeholder . When a value is passed typescript infers the exact type. That type is then enforced throughout the code. In result no need to re write the code for each type. No loss of typesafety like any.
